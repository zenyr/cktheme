# cktheme

LIVE DEMO: https://cktheme.now.sh

# Prerequisite

- `node.js` 6+
- (optional) `yarn`

# Developing

- `git clone` this repo
- `cd cktheme`
- `npm i` or `yarn`
- `npm run dev` or `yarn dev`
- open `http://localhost:8080`

# Building static files

- `npm run build` or `yarn build`
- copy `build` folder
- host on any web servers
